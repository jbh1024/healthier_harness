## 의사결정
- **테스트 추가 범위 선정**: 비즈니스 로직 복잡도/보안 영향이 높은 영역 우선
  - `AuthService`: 비밀번호 규칙·중복 이메일·자격 검증 (보안 핵심)
  - `CreditService`: 잔여 횟수 차감/복구 (크레딧 무결성 핵심)
  - `InviteCode` 엔티티: `isUsable()` 분기(만료/비활성/사용횟수/unlimited) (가입 흐름 핵심)
- **Stub 유지 결정**: Google OAuth / Passkey / FCM 는 PRD에 "외부 SDK 의존"으로 명시됨. 코드상 인터페이스·DB 스키마·API는 이미 정의됐으므로, 외부 라이브러리 추가 시 구현체 교체만으로 동작 가능. 재개발 없이 현 상태 유지.
